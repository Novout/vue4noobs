# Vue Property Decorator

No Vue-CLI é posível escolher a opção de usar o Vue com Typescript, e o Property Decorator melhora esta experiência.

Exemplo:

import { Vue, Component, Emit } from 'vue-property-decorator';

@Component
export default class He4rtComponent extends Vue {
  he4rt = 'He4rt';

  @Emit()
  Dev() {
    this.he4rt += ' Developers';
  }
};
